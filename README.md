# JAVASCRIPT TRAINING

### Scope

Where a variable can be accesed, meaning where you can use a variable.

- Inner scope has access to outer scope, but not reverse, forming **Scope Chain**
- We have **Global Scope**, **Function Scope**, and **Block Scope**.

### Hoisting

Hoisting is moving variables & functions to top of their scope.

- var is hoisted as undefined
- let, const hoisted but unitialized
- function declaration is hoisted, but not arrow functions or function expressions (func stored in var).
